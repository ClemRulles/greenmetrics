# Internationalization Plan (EN/FR) [DRAFT]

## Routing & Detection
- Localized paths: `/en/*`, `/fr/*`.
- Detection order: query `?lng=`, path, cookie `lng`, `Accept-Language` (en|fr), fallback `en`.
- Persist choice in cookie; no auto-redirect loops.

## Content Storage
- `/public/locales/{en,fr}/common.json`, `wizard.json`, `report.json`, `errors.json`, etc.
- All UI strings must live in JSON; **no literals in components**.

## Formatting
- Use `Intl` + i18next ICU for dates, numbers, percentages, and pluralization.
- Validation errors localized through Zod → i18next mapping.

## QA

- Visual spot-check: Storybook (optional) for localized components.
